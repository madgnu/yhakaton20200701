## Описание

Проект по мотивам notion

## Локальный запуск

1. Склонировать репозиторий
2. Доставить отсутствющие модули npm

    ```bash
        npm install
    ```

3. Запустить локальный сервер

    ```bash
        npm run dev
    ```
